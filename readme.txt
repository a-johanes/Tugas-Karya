Ini merupakan program kalkulator sederhana
yang menerima input berupa operasi matematika dalam bahasa indonesia
seperti "dua tambah dua" dan akan menghasilkan "dua tambah dua adalah empat".

Operator yang tersedia baru ada tiga yakni tambah, kurang dan kali.
Program ini dapat menerima input angka diantara -999 dan 999.

Untuk angka negatif cukup menambahkan kata "negatif" misalnya: 
"negatif lima kali negatif tujuh" akan menghasilakan
"negatif lima kali negatif tujuh adalah tiga puluh lima"

Program ini belum dapat memverifikasi masukan sehingga masukan harus selalu benar.

Program akan terus mengulang meminta masukan hingga masukannya adalh "-999" tanpa petik dua.

Cara penggunaannya adalah dengan save as kode raw ini dan save dengan format .java.
Pastikan sudah menginstal java jdk.
Lalu file tersebut dapat dicompile dengan compiler java dan dijalankan dari IDE,
atau menggunakan cmd dengan command sebagai berikut:
-arahkan directory cmd ke tempat file .java ini di simpan,
-ketik "javac <namafile>.java" tanpa tanda petik dua dan <namafile> diganti dengan nama file saat di save as,
-jika berhasil, ketik "java <namafile>" tanpa tanda petik dua dan tanpa .java,
-program dapat dijalankan.

Jika ada masalah dengan java dapat melihat troubleshoot dari Google.
